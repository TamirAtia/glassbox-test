# Glassbox-test



## Features

- Express
- sequelize


## Requirements

- [node](https://nodejs.org/en/)
- [git](https://www.robinwieruch.de/git-essential-commands/)

## Installation

- `git clone https://github.com/TamirAtia/glassbox-test.git`
- `cd glassbox-test/server`
- `yarn install`
- `node index.js`
